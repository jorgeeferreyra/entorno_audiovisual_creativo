### 1.3 核心竞品最新画像

#### OiiOii（上海天码形空科技）— T0 直接竞品 🆕🔴
- **发布**：2026.04.07 全球首发，已开放注册
- **官方定位**："全球首个专注于动画创作的 AI Agent 智能体"
- **架构（与本项目高度同构）**：**7 个专业 AI Agent**
  1. 艺术总监 Agent（风格统筹）
  2. 编剧 Agent（剧本/拆解）
  3. 角色设计 Agent
  4. 场景设计 Agent
  5. 动画师 Agent（视频生成）
  6. 剪辑师 Agent
  7. 音效师 Agent
- **5 大创作模式**：剧集短片、音乐概念视频、快速视频生成、漫画转视频、IP 衍生设计
- **输入→输出**：200 字故事 / 单张参考图 → 5–15 分钟内输出 1080P 完整动画短片
- **核心能力**：
  - **160+ 艺术风格模板**（动漫/电影感/赛博朋克/水墨等）
  - **全局资产记忆库**：每个角色高维特征向量，跨场景一致性
  - **底层引擎**：已直接集成 **Seedance 2.0 API**，单条剧情可输出 5 分钟完整漫剧
- **威胁等级**：🔴🔴 极高 — 这是目前发现的与 ai-comic-studio 架构最相似的直接竞品（7 Agent vs 我们 8 Agent），且已抢先接入最新引擎

#### 小云雀（字节跳动·剪映团队）— T0直接竞品 ⭐
- **发布**：2025年上线，2026.03.19发布"短剧Agent"
- **底层**：Seedance 2.0 + Seedream 5.0 + DeepSeek + 豆包大模型
- **核心能力**：10万字剧本→自动拆解→角色设计→分镜→视频→成片；支持2D/3D/准真人风格
- **标杆案例**：5人团队8天生产60集《万兽独尊》→抖音4天破亿播放
- **价格**：目前免费（每日120积分，Seedance 2.0零积分使用）
- **威胁等级**：🔴极高 — 字节生态(抖音分发)+免费策略+Seedance 2.0底层

#### Seko 2.0（商汤科技）— T0直接竞品 ⭐
- **发布**：2025.12.15
- **核心技术**：SekoIDX（负参考图角色一致性）、SekoTalk（多人唇型同步）、LightX2V（实时推理框架）
- **核心能力**：一句话→100集连续剧；智能Agent调度角色/场景/道具跨集关联；100+动画风格
- **特色**：消费级GPU上5秒生成5秒视频；适配国产芯片（寒武纪/沐曦）
- **数据**：20万+创作者，头部作品《婉心计》登顶抖音AI短剧榜
- **价格**：免费+付费（"一杯奶茶的价格生成100集"）

#### Seedance 2.0（字节跳动·即梦）— T3引擎 ⭐（含 2026.03 重大迭代）
- **发布**：2026.02.07，**2026.03 大幅扩展能力边界**
- **架构**：统一多模态音视频联合生成架构
- **多模态参考系统**：
  - **12 文件参考体系**：最多 9 张图片 + 3 段视频 + 3 段音频 + 文本提示
  - 视频参考可提取运镜/动作轨迹；音频参考用于节拍同步、对白对位、角色动作匹配
- **原生音视频联合生成**：对白 + 环境音 + 音效 + 配乐一 pass 输出
- **🆕 视频编辑能力（2026.03 新增）**：
  - **角色替换**（character replacement）
  - **内容增删**（add/delete elements in shot）
  - **视频续写 / 拼接**（extension & concatenation）
  - 引擎首次跨入"可编辑生成"阶段
- **🆕 生态分发（2026.03）**：CapCut、Dreamina、Spark、豆包、**小云雀**、**OiiOii**、**HeyGen** 全部接入
- **🆕 区域分阶段开放**：巴西 / 印尼 / 马来 / 墨西哥 / 菲律宾 / 泰国 / 越南先行
- **安全合规**：禁用真人面孔视频生成；隐形水印
- **角色一致**：身份保持系统，2-4 张多角度参考图可降低 60% 漂移
- **价格**：fal.ai API ~$0.30/秒（1080P+音频约$0.60/10秒）

#### 可灵3.0（快手）— T3引擎 ⭐
- **发布**：2026.02.05
- **架构**：Omni One（3D时空联合注意力 + 思维链推理）
- **核心能力**：原生4K/60fps；单次最长15秒；单次生成内支持最多6个镜头切换；原生多语言唇型同步
- **角色一致**：主体上传锁定面部/体态/服装/声音；跨镜头身份稳定
- **镜头控制**：pan/tilt/dolly/orbital/crane/rack focus + Motion Brush手绘运动路径
- **排名**：全球T2V Arena #1（1240 Elo）、I2V #3
- **价格**：免费66积分/天；Ultra $60/月（4K/60fps）；API $0.084-0.168/秒

#### Vidu Q3 Pro（生数科技）— T3引擎
- **发布**：2026.02初
- **核心能力**：全球首个16秒原生音视频直出（对白+音效+配乐一pass）；1080P；强角色一致性
- **排名**：全球#2（仅次可灵3.0），国内I2V Arena #1（1351 Elo）
- **价格**：竞品3-7倍便宜

#### Runway Gen-4.5 + Multi-Shot App — T1管线
- **发布**：Gen-4.5 2026.02，Multi-Shot App 2026.03.30
- **核心能力**：角色一致性Elo全球#1；Multi-Shot App单prompt→5镜头+自动运镜+对白+音效+剪辑
- **估值**：$5.3B（2026.02融资$315M）

#### PixVerse V6 + R1 — T3引擎（新锐）
- **发布**：V6 2026.03.30，R1 2026.01
- **核心能力**：V6 15秒1080P + **20+电影镜头控制**（焦距/光圈/景深/色差/暗角）；R1实时交互世界模型
- **定位**：镜头语言精细度新标杆（独步业界）

#### Higgsfield Cinema Studio 2.5 — T1管线
- **发布**：2026.03
- **核心能力**：多模型聚合(Sora/Veo/Wan/Kling)；**Soul Cast AI演员**（持久化逼真角色）；内建调色套件
- **数据**：1500万+用户，450万次生成/天，$200M ARR

## 三、SWOT 分析（2026 Q1 更新版）

### 3.1 优势 (Strengths) — 仍然成立的

| 编号 | 优势项 | 差异化程度 |
|------|--------|:---:|
| S1 | **多Agent协作 + Gate门控介入** — 全行业仅此一家有完整8 Agent分工+用户逐步介入机制 | 🟢 高 |
| S2 | **多引擎智能路由(4视频+5图片)** — 竞品多为单引擎绑定，我们可动态选最优 | 🟢 高 |
| S3 | **McKee专业叙事框架** — 三幕式、角色弧光、情绪曲线，竞品多为简单prompt | 🟡 中高 |

## 四、系统升级方案（2026.04.08 修订版 — 应对 OiiOii 同构竞品冲击）

> **本次修订核心**：应对 2026.04.07 OiiOii 同构竞品冲击。OiiOii 已抢先验证"多 Agent + Seedance 2.0 + 全局资产库 + 160 风格"组合是市场可接受形态，我们必须**最快速度追平基础设施**，并 all-in **OiiOii 没有的三项壁垒**（McKee 叙事 / 制片人闭环 / Gate 门控）。

### 4.1 战略定位修订

```
旧定位: 自建角色一致性 → 与引擎竞争
  ↓ 问题：引擎端一致性进化速度远超应用层自建

新定位: "AI导演工作台" → 利用最强引擎 + 差异化创作体验
  ↓ 核心价值：不是"谁的模型更好"，而是"谁的创作流程更专业、更可控"

**差异化锚点：**
1. **专业叙事** — McKee方法论 vs 竞品简单prompt（我们独有）

3. **多引擎路由** — 动态选最优引擎 vs 竞品绑定单一引擎（仅Higgsfield类似）

// 新增: 可灵3.0 引擎 (Kling 3.0)
interface KlingV3Options {
  duration: 3 | 5 | 10 | 15;        // 3-15秒可选
  resolution: '720p' | '1080p' | '4k'; // 原生4K支持
  fps: 30 | 60;

  nativeAudio?: boolean;               // 原生音视频一体生成

    perShotDuration?: number[];
  };
}

// 新增: Seedance 2.0 引擎
interface SeedanceV2Options {
  duration: 4 | 5 | 8 | 10 | 15;     // 4-15秒
  referenceImages?: string[];           // 最多9张图片参考
  referenceVideos?: string[];           // 最多3段视频参考(运镜轨迹提取)
  referenceAudios?: string[];           // 最多3段音频参考(节拍同步)
  nativeAudio?: boolean;                // 对白+环境音+音效+配乐一体
  cameraMotion?: CameraMotion;
}

// 新增: Vidu Q3 Pro 引擎
interface ViduQ3Options {
  duration: 4 | 8 | 16;                // 最长16秒
  nativeAudioVisual?: boolean;          // 全球首创16秒音视频直出
  subjectReference?: string[];          // 多主体参考
}

// 升级: 智能路由策略 v2
function routeVideoEngineV2(shot: ShotData): EngineRoute {
  // 需要原生音频(有台词的镜头) → 可灵3.0/Seedance 2.0/Vidu Q3 Pro
  if (shot.dialogue) {
    return { primary: 'kling3', fallbacks: ['seedance2', 'viduq3'] };
  }
  // 多角色同画面 → Seedance 2.0(@提及系统)
  if (shot.characters.length >= 2) {
    return { primary: 'seedance2', fallbacks: ['kling3'] };
  }
  // 复杂镜头运动 → 可灵3.0(Motion Brush) / PixVerse V6
  if (shot.cameraMotion?.complexity === 'high') {
    return { primary: 'kling3', fallbacks: ['seedance2'] };
  }
  // 长镜头(>10秒) → Vidu Q3 Pro(16秒) / Seedance 2.0(15秒)
  if (shot.targetDuration > 10) {
    return { primary: 'viduq3', fallbacks: ['seedance2', 'kling3'] };
  }
  // 省钱模式 → Vidu Q3 Pro(最便宜) / Veo 3.1 Lite
  if (shot.costMode === 'budget') {
    return { primary: 'viduq3', fallbacks: ['veo31lite', 'minimax'] };
  }
  // 默认: 可灵3.0(全球#1)
  return { primary: 'kling3', fallbacks: ['seedance2', 'viduq3', 'minimax'] };
}
```

## 七、核心竞争壁垒建设（修订版）

### 7.1 不要与引擎竞争，要做引擎之上的"调度者"

```
❌ 错误路线: 自建角色一致性模型 → 追不上可灵3.0/Seedance 2.0的迭代速度
✅ 正确路线: 做最快集成最新引擎的"AI导演工作台" → 引擎越强我们越强
```

### 7.2 三个不可替代的差异化壁垒

| 壁垒 | 为什么竞品难以复制 |
|------|-------------------|
| **McKee叙事引擎** | 需要深厚编剧专业知识，不是简单prompt engineering |
| **多Agent可视化协作+Gate门控** | 需要从产品理念层面重新设计交互，竞品已定型为"黑盒" |
| **7+引擎智能路由** | 需要逐一对接+维护N个API，竞品多为自研引擎无动力做路由 |

### 7.3 定位一句话

> **"最专业的AI导演工作台 — 调动全球最强AI引擎，按McKee方法论讲好你的故事"**

### 7.4 与 OiiOii 的差异化对话框（销售话术）

| 维度 | OiiOii 卖点 | 我们的差异化回应 |
|------|------------|----------------|
| 7 个 AI Agent | "全球首个动画 AI Agent" | 我们 8 个 Agent，且每个 Agent 都可被人工 Gate 介入和调整 |
| 160+ 风格模板 | 风格丰富 | 我们 v2.1 同等规模，且每个风格可由 McKee 引擎反向推荐适配类型 |
| 全局资产记忆库 | 跨场景一致 | 我们 v2.0 同步上线，且支持跨项目复用与版本管理 |
| 5 分钟生成短片 | 速度快 | 我们提供同等速度的"快速模式"，但默认推荐"专业模式"含制片人审核 |
| Seedance 2.0 集成 | 引擎最新 | 我们集成 7 个引擎，按镜头智能路由，单引擎不是优势是限制 |
| 200 字→ 5 分钟剧 | 输入门槛低 | 我们的 McKee 编剧引擎从 200 字推导出三幕结构 + 角色弧光 + 情绪曲线，深度高一个量级 |
